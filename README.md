# lab_11
# Заміна слова в реченні на символи # на C

Ця програма демонструє роботу з рядками та маніпуляції словами в реченні. Користувач вводить речення довжиною до 200 символів і номер слова, яке потрібно замінити. Програма знаходить вказане слово та замінює всі його символи на символи #, зберігаючи розташування інших слів у реченні.

Програма враховує пробіли між словами та перевіряє правильність введеного номера слова. Якщо номер слова некоректний або такого слова в реченні немає, виводиться відповідне повідомлення. В іншому випадку користувач отримує модифіковане речення з заміненим словом.

Програму можна компілювати за допомогою gcc main.c -o main і запускати командою ./main. Після запуску користувач вводить речення і номер слова, а програма виводить результат з заміною. Це дозволяє практикувати обробку рядків та індексацію символів у C.
